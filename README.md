# my_pacman
pacman using c++ &amp; ncurses.

## What is Nucurses?
Ncurses (or Ncursesw) is a library for managing screens, keystrokes, cursors, etc. in CUI.

## Usage
Build.
```sh
make
```
Run.
```sh
./pacman
```
